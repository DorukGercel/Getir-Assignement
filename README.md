# Getir Node.js Bootcamp Graduation Project

## Deployed App:
https://doruk-gercel-getir-assignement.herokuapp.com/

## How to Use:
#### First:
  ```
   npm install
   ```
#### Then:
To run the app:
   ```
   npm start
   ```
To execute tests:
   ```
   npm test
   ```
   
## Endpoint:
`POST /`
